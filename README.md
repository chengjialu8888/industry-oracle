# Industry Oracle — 行业深度认知加速器

A Mira skill that compresses a month of industry research into one hour.

## What It Does

Upload industry materials (PDF, web pages, Feishu docs), and AI guides you through a 4-step structured process:

1. **Industry Panoramic Analysis** — Paradigm positioning × Competitor deep cognition × Demand validation
2. **Core Assumption Deconstruction** — 3 assumptions that, if overturned, rewrite the industry
3. **Investor Pressure Test** — 5 top-tier investor perspectives with evidence-based answers
4. **Multi-Round Debate** — Sharpen every argument to its strongest form

Each step generates a Feishu document and pauses for user confirmation.

## Features

- **Bilingual** — Chinese + English, auto-matches user language
- **MECE Thinking, Sharp Expression** — Think comprehensively, express selectively
- **Paradigm-Sensitive** — Detects whether the industry is in stable/cracking/migration phase
- **Demand Authenticity** — Separates real use cases from capital-driven hype
- **Evidence-Driven** — All insights anchored to source materials

## Install

Copy `SKILL.md` into your Mira skill directory, or install the `.skill` package.
